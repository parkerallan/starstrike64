# Star Strike - N64

A 3D rail-shooter mecha game for Nintendo 64 using libdragon and Tiny3D.  

<img src="https://github.com/user-attachments/assets/4374e3b1-662c-44ca-983c-38e9b04929db"/>

## How to Play
Download the .z64 rom from the Releases tab and run it on an emulator like Gopher64 or for best performance, run it on a flash cart with original hardware.  

### Controls
- Directional movement uses the control stick
- A button for shooting
- B button for power slash
